This site represents a showcase of our tehnological inventions, in terms of mobile devices, from the last years. This is a personal project, in which I tried creating a useful guide in choosing a phone, linked with a big collection of phones which you can choose from.

This page won't contain much, only a summary of the other pages, features. Also on this page you can find a search bar, in the right part of the menu. The search bar is implemented to search any field in the database. (example: search for '2018' to get all the phones released in 2018, search for 'huawei' to get all the phones from that brand)

The guide page contains a small guide that might be handful when picking a phone.

The deals page contains some data filtering on the database, showcasing some maybe helpful criteria when picking a phone.

The brands page is a dynamic page and changes based on brand. You can also view all the database entries.

If you find some phones missing, you can send information about them in the propose page.

In the contact page, you will see some short summary of my work and will be able to leave any comments, regarding the site or whatnot. Suggestions are welcome.
